# الموقع التعليمي

موقع تعليمي يضم 4 صفحات رئيسية:

## الصفحات:
1. **index.html** - الصفحة الرئيسية (عرض الصفوف والمواد)
2. **profile.html** - الملف الشخصي
3. **admin.html** - قسم الإدارة
4. **study-plan.html** - تنظيم الدراسة

## كيفية التشغيل:

### الطريقة الأولى (الأسهل):
- انقر نقراً مزدوجاً على ملف `index.html` لفتحه في المتصفح

### الطريقة الثانية:
- افتح المتصفح (Chrome, Firefox, Edge)
- اضغط Ctrl+O (أو File > Open)
- اختر ملف `index.html`

### الطريقة الثالثة (للمطورين):
- استخدم Live Server في VS Code
- أو استخدم أي خادم محلي مثل Python: `python -m http.server 8000`
